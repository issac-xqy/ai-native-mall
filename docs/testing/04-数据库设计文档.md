## 4. Redis 向量存储

### 向量索引配置

| 属性 | Product知识库 | FAQ知识库 |
|------|:--:|:--:|
| 索引名 | `ai_mall_product_knowledge` | `ai_mall_faq_knowledge` |
| 维度 | 1536 | 1536 |
| 距离度量 | COSINE | COSINE |
| 索引类型 | FLAT | FLAT |
| 向量模型 | text-embedding-v4 | text-embedding-v4 |

### 向量维度兼容

系统启动时 `VectorStoreDimensionChecker` 检测现有向量维度，如配置维度(1536/external)与存量不匹配(384/ONNX)，自动清空重建。

## 5. MyBatis-Plus 约定

| 约定 | 配置 |
|------|------|
| 主键策略 | `IdType.AUTO` (数据库自增) |
| 逻辑删除 | `deleted` 字段, 0-未删 1-已删 |
| 自动填充 | `create_time` (INSERT), `update_time` (INSERT_UPDATE) |
| 下划线转驼峰 | `map-underscore-to-camel-case: true` |
| 分页插件 | `PaginationInnerInterceptor` (MySQL) |
