## 1. 管理端 (web-admin) — 端口 3000

### 1.1 页面结构

```
/admin (管理后台布局)
├── Sidebar 侧边栏 (固定左侧)
│   ├── Logo + 系统名称
│   └── 菜单项 (el-menu vertical)
│       ├── 📊 Dashboard
│       ├── 📦 商品管理
│       │   ├── 商品列表
│       │   └── 新增商品
│       ├── 📂 分类管理
│       ├── 🤖 AI商品助手
│       ├── 💬 评论分析
│       ├── 📈 AI监控
│       ├── 📚 知识库
│       └── 📋 订单管理
├── Header 顶栏
│   ├── 面包屑导航
│   └── 用户头像 + 退出
└── Main 内容区
    └── <router-view />
```

### 1.2 页面清单

| 路由 | 页面组件 | 功能描述 |
|------|----------|----------|
| `/` | `Dashboard.vue` | 今日销售/订单/AI对话/满意度 4个统计卡片 |
| `/products` | `ProductList.vue` | 商品表格(名称/分类/价格/库存/状态), 搜索/筛选/操作 |
| `/product/create` | `ProductForm.vue` | 新建商品表单(名称/分类/价格/库存/图片/描述/规格) |
| `/product/edit/:id` | `ProductForm.vue` | 编辑商品(同form, 预填数据) |
| `/categories` | `CategoryList.vue` | 分类树形表格(名称/排序/状态), 增删改 |
| `/product-ai` | `ProductAI.vue` | AI生成SEO标题 + 商品描述 |
| `/comment` | `CommentAnalysis.vue` | 评论列表 + AI情感分析结果 |
| `/ai-monitor` | `AiMonitor.vue` | AI调用统计图表(次数/响应时间/错误率/小时趋势) |
| `/knowledge` | `KnowledgeBase.vue` | 文档列表(名称/类型/状态), 上传/删除/重新向量化 |
| `/orders` | `OrderList.vue` | 订单表格(订单号/用户/金额/状态), 筛选/发货操作 |

### 1.3 组件

| 组件 | 用途 |
|------|------|
| `ImageUpload.vue` | 图片上传 (el-upload + 预览 + 大小限制) |

## 2. 用户端 (web-client) — 端口 3001

### 2.1 页面结构

```
/ (商城主布局)
├── Header 顶栏 (固定顶部, 白色背景, 阴影)
│   ├── Logo "智能商城"
│   ├── 导航菜单 (el-menu horizontal)
│   │   ├── 首页
│   │   ├── 全部商品
│   │   ├── 购物车 (带badge数量)
│   │   ├── 我的订单
│   │   ├── 我的钱包
│   │   ├── AI客服
│   │   └── 个人中心
│   ├── 搜索框 (el-input + Search图标)
│   └── 用户区
│       ├── 已登录: 头像 + 用户名下拉 (个人中心/订单/退出)
│       └── 未登录: 登录按钮
├── Main 内容区 (灰色背景 #f5f7fa)
│   └── <router-view /> (ErrorBoundary包裹)
└── Footer 页脚
    └── "© 2026 AI-Native Smart Mall - Powered by xqy"

/login (登录页面 - 全屏独立布局, 无Header/Footer)
└── Login.vue (居中卡片式登录/注册表单)
```

### 2.2 页面清单

| 路由 | 页面 | 鉴权 | 功能 |
|------|------|:--:|------|
| `/login` | `Login.vue` | 否 | 登录/注册切换, 表单验证 |
| `/` | `Home.vue` | 否 | AI轮播推荐 + 热销商品 + 好评商品 |
| `/products` | `Products.vue` | 否 | 商品列表(分类筛选+搜索+排序+分页) |
| `/product/:id` | `ProductDetail.vue` | 否 | 商品详情(图片/价格/规格/描述/AI推荐) |
| `/cart` | `Cart.vue` | 否 | 购物车(选择/数量/总价/去结算) |
| `/checkout` | `Checkout.vue` | 是 | 结算(选择地址/确认订单) |
| `/payment` | `Payment.vue` | 是 | 钱包支付 |
| `/orders` | `Orders.vue` | 是 | 订单列表(状态筛选/确认收货/取消) |
| `/profile` | `Profile.vue` | 是 | 个人信息(昵称/邮箱/头像编辑) |
| `/wallet` | `Wallet.vue` | 是 | 钱包(余额/充值/充值记录/消费记录) |
| `/ai-chat` | `AIChat.vue` | 否 | AI客服(聊天界面/SSE流式) |
| `/upload-demo` | `UploadDemo.vue` | 否 | 文件上传演示 |
| `/:pathMatch(.*)*` | `NotFound.vue` | 否 | 404页面 |

### 2.3 组件

| 组件 | 用途 |
|------|------|
| `AICarousel.vue` | AI推荐轮播(自动播放5s, 推荐理由, 置信度) |
| `ErrorBoundary.vue` | 错误边界(捕获组件异常, 显示重试/回首页) |
| `ImageUpload.vue` | 图片上传(拖拽+预览+压缩) |

### 2.4 Store 状态管理

| Store | 职责 |
|-------|------|
| `stores/user.ts` | 登录状态/Token管理/用户信息/localStorage持久化 |
| `stores/cart.ts` | 购物车增删改/全选/总计 |

### 2.5 鉴权流程

```
用户访问需要鉴权的路由(/checkout, /payment, /orders, /profile)
    │
    ▼
router.beforeEach 检查 token
    │
    ├── 无 token → 重定向 /login
    └── 有 token → 正常进入
            │
            ▼
        App.vue 显示用户信息
            │
            ▼
        7天过期 → 自动登出
```

## 3. 设计规范

### 3.1 色彩系统

| 用途 | 色值 |
|------|------|
| 主色调 | `#409EFF` (Element Plus 默认蓝) |
| 成功/上架 | `#67C23A` |
| 警告/待处理 | `#E6A23C` |
| 危险/删除 | `#F56C6C` |
| 背景灰 | `#f5f7fa` |
| 文字黑 | `#303133` |
| 次要文字 | `#909399` |

### 3.2 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| 桌面端 | ≥1200px | 完整布局 |
| 平板端 | 768-1199px | 侧边栏折叠 |
| 移动端 | <768px | 汉堡菜单 + 单列 |

### 3.3 文案规范

| 场景 | 示例 |
|------|------|
| 按钮 | 登录/注册/加入购物车/立即购买/确认支付 |
| 提示 | 操作成功/请输入用户名/余额不足 |
| 状态 | 待支付/已支付/已发货/已完成/已取消 |
| 空状态 | 购物车空空如也~/暂无订单 |
